# td

td is a To-do TUI app written in golang

## Installation

### macOS
```bash
brew tap voioo/td
brew install td-tui
```

### Arch Linux
```bash
yay -S td-tui
```
or
```bash
paru -S td-tui
```
or
```bash
git clone https://aur.archlinux.org/td-tui.git
cd td-tui
makepkg -si
```
